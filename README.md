# Caching NodeJS :technologist:

Esse projeto consiste em um demonstrativo de uma aplicação nodejs
utilizando o conceito de caching.

## Iniciando o projeto
    - docker-compose up -d --build
    - npm run loadtest:nocache
    - npm run loadtest:cached

## Features Utilizadas
    - MongoDB
    - Redis
    - NodeJs
